Example: P&L Extraction Script (Python)

import pandas as pd

# load P&L (example)
df = pd.read_csv('pl.csv')
# aggregate by segment
summary = df.groupby('segment').agg({'revenue':'sum','expense':'sum'})
summary['margin'] = (summary['revenue'] - summary['expense']) / summary['revenue']
print(summary)

Financial Formula

Return on Equity (ROE):

\(\mathrm{ROE} = \dfrac{\mathrm{Net\ Income}}{\mathrm{Shareholders'\ Equity}}\)

DuPont decomposition:

\(\mathrm{ROE} = \mathrm{Net\ Profit\ Margin} \times \mathrm{Asset\ Turnover} \times \mathrm{Equity\ Multiplier}\)
